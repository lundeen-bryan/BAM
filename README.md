# BAM (Bryan's Adding Machine)

BAM (Bryan's Adding Machine) is a C# WinForms application designed to emulate the behavior of a traditional desktop adding machine.

This project is inspired by **Mark's Adding Machine (MAM)** and aims to recreate and better understand its functionality while building a clean, maintainable architecture from the ground up.

## Project Status

This project is currently **under active development**.

Features are being implemented incrementally, with a strong focus on:

- Accurate adding machine behavior
- Clean separation of concerns
- Maintainable and testable code
- Reusable core logic (CalculatorEngine)

## Goals

- Recreate the behavior of a traditional adding machine
- Implement a robust calculation engine independent of the UI
- Provide a structured "tape" system for tracking calculations
- Serve as a learning project for improving C# and application design skills

## Technology

- Language: C#
- Framework: .NET (WinForms)
- Architecture: Layered (Forms, Handlers, Services, Models)
